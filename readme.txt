add a "vids" folder, and place all videos inside of it
add a "lib" folder, download mpv-dev from here: https://sourceforge.net/projects/mpv-player-windows/ and extract the files to the lib folder

Space = pause/play
Escape = quit
Enter = restart videos
f = toggle fullscreen
l = toggle loop
b = toggle border
s = toggle window snap
c = toggle on screen controller
k = toggle stay open on finish
